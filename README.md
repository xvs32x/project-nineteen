# Front

Содержит Web приложение для отображения в браузере.
Для запуска:
- Установить NODEJS 18 (https://nodejs.org)
- Перейти в папку front
- Выполнить `npm install`
- Выполнить `npm start`
- Открыть в браузере http://localhost:4200
